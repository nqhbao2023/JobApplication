📘 JOB4S – HƯỚNG DẪN CÀI ĐẶT MÔI TRƯỜNG DỰ ÁN (React Native + Expo + Firebase)

───────────────────────────────
I. CÀI ĐẶT CƠ BẢN
───────────────────────────────
1️⃣ Cài Node.js (phiên bản LTS)
   - Truy cập: https://nodejs.org/
   - Cài bản **Node 20.x hoặc 22.x LTS**
   - Kiểm tra:
        node -v
        npm -v
        npx -v

2️⃣ Cài Git
   - Link: https://git-scm.com/downloads
   - Kiểm tra:
        git --version

3️⃣ Clone lại project
   - Mở PowerShell:
        cd Documents\GitHub
        git clone https://github.com/ten-tai-khoan/JobApplication.git
        cd JobApplication

───────────────────────────────
II. CÀI ĐẶT EXPO / NPM MODULES
───────────────────────────────
1️⃣ Cài Expo CLI (local)
        npm install -g expo-cli

2️⃣ Cài các gói trong project
        npm install

3️⃣ Nếu bị lỗi module:
        npm cache clean --force
        npm install

───────────────────────────────
III. CÀI ĐẶT ANDROID ENVIRONMENT
───────────────────────────────
1️⃣ Cài Android Studio
   - Link: https://developer.android.com/studio
   - Trong “More Actions” → SDK Manager → chọn:
        ✅ Android SDK Platform 34
        ✅ Android SDK Build-Tools 34.x
        ✅ Android Emulator
        ✅ Android SDK Command-line Tools (latest)
        ✅ Intel x86_64 Emulator Accelerator (HAXM)

2️⃣ Tạo máy ảo (AVD)
   - Open Device Manager → Create Virtual Device:
        📱 Device: Pixel 8
        🧩 API Level: 34 (Android 14, Google Play)
        🧠 RAM: 2048 MB
        ⚙️ ABI: x86_64
        ✅ Finish

3️⃣ Kiểm tra adb
        adb version
        adb devices

───────────────────────────────
IV. CÀI ĐẶT JAVA (CHO GRADLE)
───────────────────────────────
1️⃣ Tải JDK 17 (Temurin)
   - Link: https://adoptium.net/
   - Cài bản “Temurin 17 (x64)”

2️⃣ Tạo biến môi trường:
   Mở: Edit the system environment variables → Environment Variables

   **User Variables:**
      JAVA_HOME = C:\Program Files\Eclipse Adoptium\jdk-17.0.16.8-hotspot
      ANDROID_HOME = C:\Users\Admin\AppData\Local\Android\Sdk

   **Thêm vào Path:**
      %JAVA_HOME%\bin
      %ANDROID_HOME%\platform-tools
      %ANDROID_HOME%\emulator

3️⃣ Kiểm tra:
        java -version
        javac -version
        echo $env:JAVA_HOME
        echo $env:ANDROID_HOME

───────────────────────────────
V. KIỂM TRA VS CODE / TYPESCRIPT
───────────────────────────────
1️⃣ Mở VS Code → terminal
2️⃣ Cài dependencies nếu cần:
        npm install --save-dev typescript @types/react @types/react-native
3️⃣ Kiểm tra tsconfig.json:
        "extends": "expo/tsconfig.base",
        "compilerOptions": {
            "strict": true,
            "baseUrl": ".",
            "paths": { "@/*": ["src/*"] }
        }

───────────────────────────────
VI. CHẠY PROJECT
───────────────────────────────
1️⃣ Khởi động Metro bundler:
        npx expo start -c

2️⃣ Khi menu hiện ra:
        → Nhấn phím **a** để mở Android emulator
        → Nhấn **r** để reload app

───────────────────────────────
VII. LỆNH HỮU ÍCH KHÁC
───────────────────────────────
🧹 Xóa cache & node_modules:
    rmdir /s /q node_modules
    del package-lock.json
    npm install

📦 Build app native:
    npx expo run:android

🧱 Reset toàn bộ:
    npm run reset-project

───────────────────────────────
VIII. KIỂM TRA SAU CÀI ĐẶT
───────────────────────────────
✅ Node.js hoạt động: node -v
✅ npm hoạt động: npm -v
✅ npx hoạt động: npx -v
✅ Java hoạt động: java -version
✅ Android SDK OK: adb devices
✅ Expo chạy được: npx expo start -c
✅ Emulator mở được: Pixel 8 API 34 hiển thị app Job4S

───────────────────────────────
✨ GHI CHÚ QUAN TRỌNG
───────────────────────────────

      Set-ExecutionPolicy Unrestricted -Scope CurrentUser
- Sau khi thay đổi biến môi trường → **đóng và mở lại PowerShell**
